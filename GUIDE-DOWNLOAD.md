## 4. Conversão para Web Fonts

Após baixar todos os arquivos `.ttf` ou `.otf`, você precisa convertê-los para formatos web (WOFF2 e WOFF).

### Opção 1: Transfonter (Recomendado) 🌟

**Link:** https://transfonter.org

**Passo a passo:**
1. Acesse https://transfonter.org
2. Clique em **"Add fonts"**
3. Selecione TODOS os arquivos `.ttf` de uma vez (Ctrl/Cmd + clique)
4. Configure:
   - ✅ **WOFF2** (marcado)
   - ✅ **WOFF** (marcado)
   - ✅ **TTF** (opcional - manter original)
   - ✅ **Base64 encode** (desmarcado)
   - ✅ **Fix vertical metrics** (marcado)
   - ✅ **Add local() rule** (opcional)
5. Clique em **"Convert"**
6. Baixe o arquivo ZIP
7. Extraia e organize nas pastas corretas

**Resultado:**
- Você receberá `.woff2`, `.woff` e um arquivo `stylesheet.css` (pode ignorar, já temos o `fonts.css`)

---

### Opção 2: CloudConvert

**Link:** https://cloudconvert.com/ttf-to-woff2

**Passo a passo:**
1. Acesse o link acima
2. Faça upload dos arquivos `.ttf`
3. Escolha formato de saída: **WOFF2**
4. Clique em **"Convert"**
5. Baixe os arquivos convertidos
6. Repita para converter para **WOFF**

---

### Opção 3: Linha de Comando (Avançado)

Se você tem Python instalado:

```bash
# Instalar fonttools
pip install fonttools brotli

# Converter TTF → WOFF2
for file in *.ttf; do
  python -m fontTools.ttLib.woff2 compress "$file"
done

# Converter TTF → WOFF (requer sfnt2woff)
for file in *.ttf; do
  sfnt2woff "$file"
done
```

## 🆘 Problemas Comuns

### ❌ "Não encontrei Ofissina ou Bolupisang"

**Solução:**
- Use as alternativas do Google Fonts:
  - **Ofissina** → **Offside**
  - **Bolupisang** → **Indie Flower**
- Ambas têm licença SIL OFL 1.1 (uso comercial permitido)

### ❌ "Conversão para WOFF2/WOFF falhou"

**Solução:**
- Tente outro conversor (Transfonter → CloudConvert)
- Verifique se o arquivo `.ttf` original não está corrompido
- Re-baixe a fonte do Google Fonts

### ❌ "Arquivo muito grande"

**Solução:**
- Use apenas formatos `.woff2` e `.woff` (ignore `.ttf` para web)
- Considere subsetting (remover caracteres não utilizados)
- Variable fonts são geralmente menores que múltiplos pesos

### ❌ "Licença não está clara"

**Solução:**
- Fontes do Google Fonts: sempre SIL OFL 1.1 (uso comercial OK)
- Fontes de outros sites: leia o `LICENSE.txt` ou página de download
- Em dúvida: use apenas fontes do Google Fonts
